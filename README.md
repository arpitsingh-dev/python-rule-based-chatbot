# Rule-Based Personal Chatbot (Python)

This project is a simple rule-based personal chatbot built using Python basics.

## Concepts Used
- Python functions
- Loops
- Conditional statements
- Dictionaries
- Date and time module
- User input and output

## Features
- Greets user based on current time
- Responds to basic predefined questions
- Continuous interaction until user exits
